# 문서 목차

이 프로젝트의 운영/개발 문서는 모두 이 디렉토리에 모아두었습니다.

## 문서 목록

| 문서 | 용도 |
|------|------|
| `01_시스템_소개.md` | 시스템 목적, 아키텍처, 데이터 흐름, 핵심 기능 |
| `02_운영_가이드.md` | 환경 변수, 실행 명령, 점검/백업/장애 대응 |
| `03_프로젝트_구조.md` | 디렉토리 구조와 주요 모듈 분석 |
| `04_도커_배포_가이드.md` | Dockerfile, Compose 서비스, 스케줄러 운영 방식 |

## 읽는 순서

1. 시스템을 이해하려면 `01_시스템_소개.md`
2. 실행하거나 운영하려면 `02_운영_가이드.md`
3. 코드를 수정하려면 `03_프로젝트_구조.md`
4. 컨테이너 배포가 필요하면 `04_도커_배포_가이드.md`
